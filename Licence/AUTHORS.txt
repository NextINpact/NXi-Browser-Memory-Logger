AUTHORS for PC INpact Browser Memory Logger

PC INpact Browser Memory Logger est une application vous permettant de connaître la consommation mémoire des navigateurs fonctionnant sur votre machine.
Vous pouvez aussi effectuer un log au sein de fichiers CSV afin d'étudier l'évolution de cette valeur dans le temps.

Projet GitHub : https://github.com/PCINpact/PCi-BML
